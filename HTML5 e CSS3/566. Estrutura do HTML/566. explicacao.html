Anatomia e Estrutura do HTML

Tags e Sintaxe

  • Tag com corpo → ‹escreve› Conteúdo ‹/escreve›
    Abertura da Tag ‹ › → Fechamento da Tag ‹/›.
  • Tag sem corpo (Self-closing) → ‹br› ou ‹br /›
    Utilizadas para elementos que não envolvem conteúdo, como quebras de linha ou imagens.
  • Tag com atributo → ‹tag atributo="valor"›
    Exemplo: ‹escreve atributo="&larr"›. Atributos fornecem informações extras para a tag.

Estrutura Básica

No editor (como VS Code), utilizamos o atalho "!" para gerar a estrutura inicial do HTML5 automaticamente.

  • ‹!DOCTYPE html› → Informa para o navegador que este é um documento HTML5.
  • ‹html lang="pt-BR"› → Tag Raiz. Tudo o que faz parte do site deve estar dentro dela.
    O atributo lang="pt-BR" informa que o conteúdo principal está em português do Brasil.
  • ‹head› → Local onde ficam os Metadados (configurações e informações do site).
    • ‹meta charset="UTF-8"› → Garante que acentuações (á, õ, ç) e caracteres especiais apareçam corretamente.
    • ‹meta name="viewport" content="width=device-width, initial-scale=1.0"› → Essencial para Responsividade (ajusta o site no celular).
    • ‹title› Document ‹/title› → Define o título que aparece na aba do navegador.
    ‹/head› → Fechamento das configurações.
  • ‹body› → O "Corpo" do site. Tudo o que for digitado aqui aparecerá para o usuário final.
    • ‹p› → Tag usada para criar parágrafos de texto.
    ‹/body› → Fechamento do conteúdo visível.
  • ‹/html› → Fechamento total do documento.
Dica de Hierarquia: O HTML funciona como uma árvore (DOM). Uma tag deve ser fechada antes da tag "pai" que a envolve.
Exemplo: O ‹title› deve fechar antes do ‹/head›.
