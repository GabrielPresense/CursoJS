2. IDs e classes

AbreviaçãoExpande para
header#top<header id="top"></header>
section.content<section class="content"></section>
div#app.container<div id="app" class="container"></div>

3. Hierarquia / Aninhamento

AbreviaçãoExpansão
nav>ul>li*4>aum <nav> com uma lista de 4 links
header>h1+pum <header> com título e parágrafo

4. Irmãos (siblings)

AbreviaçãoExpansão
h1+h2+h3três tags em sequência
div+p+imguma div, um p e uma img

5. Repetição, contadores e atributos

AbreviaçãoExpansão
li.item$*55 itens com classes item1 a item5
a[href="#"]<a href="#"></a>
img:srcexpande para <img src=""> (dependendo do editor)

6. Texto padrão

AbreviaçãoResultado
loremGera texto Lorem ipsum
lorem20Gera 20 palavras de Lorem ipsum

Dicas rápidas

  • No VS Code, pressione Tab após digitar a abreviação (ou Ctrl+E / Ctrl+Alt+Enter dependendo da configuração).
  • Use ^ para voltar ao elemento pai (ex: div>ul>li^p gera <p> como irmão de div).
  • Combine com snippets do VS Code para acelerar ainda mais.
